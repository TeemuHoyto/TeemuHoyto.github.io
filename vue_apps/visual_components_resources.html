Visual Components Resources
  • Visual Components Resources combines a blog with case studies, ebooks, events and webinars.
  • All data is managed with Vue.js
  • Vue router adds main categories as links that are used in the mega menu navigation and sent to possible clients as sales material.
  • Events and webinars are displayed differently than other posts. Back-end magic checks dates of events and webinars and determines their status as future or past events.
  • Rest api has been expanded with featured image and ACF field. Only required fields are optained from the Rest api.
