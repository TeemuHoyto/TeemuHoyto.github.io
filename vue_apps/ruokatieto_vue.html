Ruokatieto

Ruokatieto tuotehaku
  • Ruokatieto Yhdistys (Finfood – Finnish Food Information) site.
  • An app that allows users to see companies and products that have Hyvää suomesta label.
  • User can sort, search and see companies and their products.
  • The data is originally fetched from an external API by an integration I made.
  • Vue fetches the data using Axios from a custom rest endpoint.
