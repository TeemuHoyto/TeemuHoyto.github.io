Grafia

Grafia site
  • I was lead developper in this project.
  • Grafia is a Association of Visual Communication Designers in Finland.
  • The site has a public side and an itranet for members.
  • The site has an integration to gruppo to fetch and update user data.
  • I made a vue app that uses the integration to fetch user data and update the data user wishes to change.
  • Users can view blog posts and news and if user logs in they have another set for members.
  • Users can add posts and events to the event calendar.
  • I also made regular development and the members vue app.
  • Admin can change the colors of the site to change the style of the Luonnos pages
  • A superb project that turned out well.
