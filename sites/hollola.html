Hollola

Hollola site
  • I was lead developper in this project.
  • Hollola is a municipality in southern Finland.
  • This site contains a PTV integration.
  • PTV stands for palvelutietovaranto that muncipalities and cities can use.
  • I made the blocks that use the PTV data in the pages. If the data in PTV changes the site data updates.
  • The site has seventy people updating the data and I hade the plesure to teach them how to use the site.
  • Contacts search allows people to search contacts by name, tittle or location.
  • RSS feeds are parsed to the page and shown to the user here.
  • Basic super fast vue apps to fetch posts.
