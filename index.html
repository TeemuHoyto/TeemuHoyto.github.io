Sympa marketplace:

Sympa marketplace

  • This is a Woocommerce site and Vue is used to display the products on this page. Vue router adds URL:s to selections that marketing can give to clients and use as a sales tool.

Sympa events and webinars:

Sympa events and webinars
  • This simple vue app gets all CPT Events and webinars posts from all languages.
  • All data management is handelled with Vue.js
  • Posts can be filterred by category and by language.
  • App checks the lang of of the website and gets the posts of that language first.
  • New routes have been added in to the wp rest api for Events and webinars postype.
  • Images and ACF field have been added to the rest api to be used in front end.
