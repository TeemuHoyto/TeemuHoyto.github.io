Vue apps

Sympa careers:

Sympa careers
  • This app gets its data from recruitee api.
  • All data management is handelled with Vue.js
  • Vue router is used to create a URL for every spesific posting.
  • Social media shares share the URL of the specific job.
  • Job ids are derived from recruitee api.
  • If job is no longer available the link vill open a popup stating that to the user.

Sympa marketplace:

Sympa marketplace
  • Vue app + Vue router
  • This is a Woocommerce site and Vue is used to display the products on this page. Vue router adds URL:s to selections that marketing can give to clients and use as a sales tool.
  • Vue router uses history mode and back button allows user to see previous "page" and it will not exit the app.

Sympa events and webinars:

Sympa events and webinars
  • This simple vue app gets all CPT Events and webinars posts from all languages.
  • All data management is handelled with Vue.js
  • Posts can be filterred by category and by language.
  • App checks the lang of of the website and gets the posts of that language first.
  • New routes have been added in to the wp rest api for Events and webinars postype.
  • Images and ACF field have been added to the rest api to be used in front end.

Visual components Resources:

Visual components Resources
  • VIsual components Resources combines a blog with case studies, ebooks, events and webinars.
  • all data is managed with vue.js
  • Vue router adds main categories as links that are used in the mega menu navigation and sent to possible clients as sales material.
  • Events and webinars are displayed differently than other posts. Backend magic checks dates of events and webinars and determins their status as future or past events.
  • Rest api has been expanded with featured image and ACF field. Only reguired fields are optained from the rest api.
