Vuodenhuiput integration to grafia

vuodenhuiput
  • User can login with Grafia sites login and password.
  • A Vue app has a login form and the data is sanitized before it is send to a rest endpoint.
  • The password is encrypted and a call is generated to a secure api that wants api key and the login data.
  • If the check passes Grafia site returns users Gruppo ID and name.
  • After succesfull api call user can add a password and use the vuodenhuiput site.
