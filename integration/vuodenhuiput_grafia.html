Vuodenhuiput integration to grafia

vuodenhuiput
  • User can login with Grafia sites login and password.
  • The integration fetches login data from Grafia site and generates a user to Vuoden huiput.
  • Integration incorporates Vue app that has a login form. As the user fills the form and submits it the data is sanitized and it is send to a rest endpoint.
  • The password is encrypted in the wp rest endpoint and a call is generated to a secure api that wants api key and the login data.
  • If the checks passes the Grafia site returns users Gruppo ID and name.
  • After succesfull api call user can add a password and use the vuodenhuiput site.
