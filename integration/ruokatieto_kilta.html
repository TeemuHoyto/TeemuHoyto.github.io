Ruokatieto
  • Plugin that gets 150 companies and 12000 products from Kilta API.
  • The data is fetched once per day.
  • If that data exists the fields are updated else the data is created.
  • If data is removed from the API it is deleted from the site also.
  • Images are eather base64, synkka images or direct URL images.
  • Some of the images are in TIFF format and the plugin checks for that.
  • Tiff images are converted to JPEG and made smaller before being saved to the media library.
  • All images are minified to 600px width before saving.
